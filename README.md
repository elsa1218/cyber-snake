# 🐍 赛博贪吃蛇 | Cyber Snake

赛博朋克风格的贪吃蛇游戏，带有表情包皮肤系统和特殊食物效果！

## 🎮 特性

- 🎨 **赛博朋克美学** - 霓虹配色 + 动态效果
- 🎭 **5 种蛇皮肤** - 机器人/经典/彩虹/火焰/幽灵
- 🍎 **5 种食物类型** - 普通/双倍/加速/减速/宝石
- ⚡ **效果系统** - 双倍分数/速度变化
- 🏆 **排行榜** - 本地保存 Top 10 成绩
- 🎵 **音效系统** - 可开关
- ⏸️ **暂停功能** - 随时暂停继续

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行游戏

```bash
python snake.py
```

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| **SPACE** | 开始游戏（主菜单） |
| **↑↓←→** | 移动蛇 |
| **1-5** | 切换蛇皮肤 |
| **P** | 暂停/继续 |
| **R** | 重新开始（游戏结束时） |
| **H** | 查看排行榜 |
| **M** | 音效开关 |
| **ESC** | 退出游戏 |

## 🎭 蛇皮肤

| 按键 | 名称 | 外观 |
|------|------|------|
| 1 | Cyber | 🤖 赛博机器人 |
| 2 | Classic | 🐍 经典绿蛇 |
| 3 | Rainbow | 🦄 彩虹独角兽 |
| 4 | Fire | 🔥 火焰 |
| 5 | Ghost | 👻 幽灵 |

## 🍎 食物类型

| Emoji | 名称 | 分数 | 效果 |
|-------|------|------|------|
| 🍎 | 普通 | 10 | 无 |
| ⭐ | 双倍 | 20 | 双倍分数 10 秒 |
| ⚡ | 加速 | 15 | 速度提升 8 秒 |
| 🐌 | 减速 | 15 | 速度降低 8 秒 |
| 💎 | 宝石 | 50 | 直接加 50 分 |

## 🏆 游戏机制

- 每吃一个食物得 10 分（基础）
- 每 50 分升一级，速度加快
- 撞墙或撞自己游戏结束
- 成绩自动保存到排行榜
